# Игровой процесс

День идет с 08:00 до 18:00. Время идет само, но игру можно поставить на паузу.

## 1. Задачи появляются в бэклоге

Бэклог - это список входящих задач. У задач есть дедлайн. Если дедлайн близко, задача становится опаснее для бизнеса.

## 2. Перетащи задачу в работу

Задача в колонке "В работе" еще не готова. На нее нужно назначать людей из команды.

## 3. Назначай людей на подзадачи

Перетащи персонажа на карточку задачи. Игра сама выберет подходящую открытую подзадачу для этого персонажа.

Если человек работает не по своей сильной роли, он может сделать хуже и сильнее устать.

## 4. Переложи готовую задачу в "Готово"

Когда задача кажется достаточно чистой, перетащи ее в "Готово". До 18:00 ее еще можно вернуть в работу, но это стоит доверия.

## 5. В 18:00 задачи уходят в прод

Только после релиза задача реально влияет на бизнес. Чистые задачи помогают. Грязные задачи могут создать хвосты на следующий день.

## 6. Утром смотри последствия

Утренний разбор показывает, что случилось после вчерашнего релиза. Если вчера выпустили риск, сегодня может появиться новая задача-последствие.
